# Refloow-AirQuality
Professional Air quality sensor monitoring dashboard with deep analytics, historical insights &amp; data visualization 

This tool is for sensor.community or other independent air quality measurement stations, helping to capture display & visualize large amounts of data gathered by each individual sensor.
