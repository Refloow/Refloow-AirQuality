# Refloow-AirQuality
Hyper-Local Sensor Dashboard & Telemetry Hub Professional Air quality sensor monitoring dashboard with deep analytics, historical insights &amp; data visualization 

This tool is for sensor.community or other independent air quality measurement stations, helping to capture display & visualize large amounts of data gathered by each individual sensor.
Its hosted locally on the network where sensor is at, it captures data sensor broadcasts, saving it in the database, and allowing easy visualisation of the captured data, AQI score at the time, and other variables captured by sensors, such as PM 2.5, 10, 1 qir quality data, temperature, humidity & air pressure bundled as a user friendly plug and play out of the box desktop application
